# 86duino plant IOT system
<br>
<br>
這個專案是利用86Duino zero所做的一個IOT系統。
使用了L86duntu作業系統，達到自動澆水以及寄信提醒的功能。

## I. File list
-------------
	1. \UI_display
		這個UI_display是使用SDL來繪製圖形介面，接上USB display後就可以在上面直接看到所有的數值，並可以澆水。
		
	2. \plant
		我們使用了MRAA來轉換C/C++的函式到JavaScript。
		我們也建立了一個網站在L86duintu上，你可以在上面看到所有的數值(例如：濕度、水量、溫度...等)，
		也有即時網路攝影機來讓你隨時觀看並照顧你的盆栽。
